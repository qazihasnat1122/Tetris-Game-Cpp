# Tetris-Game-Cpp
• Classic Tetris game implemented in C++ using SFML
• Features:
  - Piece rotation & movement
  - Collision detection
  - Dynamic difficulty scaling
  - Pause & menu system
  - High score saving (file handling)
• Technologies:
  - C++
  - SFML 3.x
• Concepts:
  - OOP
  - Game loops
  - Event-driven programming
